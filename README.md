# ghesmanage [![PkgGoDev](https://pkg.go.dev/badge/github.com/hnakamur/ghesmanage)](https://pkg.go.dev/github.com/hnakamur/ghesmanage)

Package ghesmanage provides an API client for calling a subset of [Manage GitHub Enterprise Server API](https://docs.github.com/ja/enterprise-server@3.17/rest/enterprise-admin/manage-ghes?apiVersion=2022-11-28).
